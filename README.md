# SaaS Catalog

Interactive SaaS catalog with vanilla JS, Gulp, and backend CRUDs.

## Overview

This project builds an interactive catalog reminiscent of the early WWW — no framework was used. It implements CRUDs and Views in the backend with Gulp, Manifest, Modernizr, and jQuery in the frontend.

## Stack

- Backend: Node.js with CRUDs
- Frontend: jQuery, Gulp, Modernizr
- CSS: Custom styles
- No frameworks

## Features

- SaaS product listing
- CRUD operations
- Search and filtering
- Responsive design

## Getting Started

```bash
npm install
gulp serve
```
